Sobre o Projeto

Este projeto de estação meteorológica IoT foi desenvolvido como iniciação científica, integrando conhecimentos de programação, eletrônica, meteorologia e análise de dados. A estação utiliza sensores conectados a um microcontrolador ESP32 para coletar dados meteorológicos em tempo real e disponibilizá-los através de uma interface web moderna, para o Centro Universitario Eduvale Avaré.

ESP32 Python FastAPI MySQL HTML/CSS/JS DHT22
